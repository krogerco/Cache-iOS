Words

## Requirements

- Xcode 14.0+
- Swift 5.5+

## Installation

The easiest way to install Cache is by adding a dependency via SPM.

```swift
        .package(
            name: "Cache",
            url: "https://github.com/krogerco/Cache-iOS.git",
            .upToNextMajor(from: Version(1, 0, 0))
        )
```
